# Private Train Assistant

一個用於記錄與安排重訓菜單的 web application。

This project is designed to be developed with AI coding agents such as Claude Code.

---

# Core Principles

1. Follow modular architecture.
2. Avoid unnecessary dependencies.
3. Prefer simple and maintainable code.
4. Each module should have a single responsibility.
5. Files should generally stay under 300 lines.
6. Avoid using `any` type unless absolutely necessary.
7. Do not modify code directly unless explicitly instructed; only provide suggestions.
8. Before providing suggestions involving specific API usage, component props, or version-specific syntax, verify with `/context7` for the latest documentation.

---

# Technology Stack

**Type**

Web App

**Language**

TypeScript

**Testing**

Vitest (unit)

# Commands

- **Install deps**: `npm install`
- **Run**: `npm run dev`
- **Test**: `npm run test`
- **Build**: `npm run build`
